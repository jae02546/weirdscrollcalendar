変なスクロールカレンダー（Weird Scroll Calendar）

プライバシーポリシー

[個人情報、機密情報について]
本アプリでは個人情報、機密情報にはアクセスしておりません。
本アプリは日本国内で使用されることを想定しております。

[祝日の取得について]
本アプリでは祝日の取得に Holidays JP API を使用しております。
詳細につきましては以下をご確認ください。
https://holidays-jp.github.io/

[広告について]
本アプリでは広告配信にAdMob(Google Inc.)を使用しております。
AdMobがご利用者様の情報を自動取得する場合がございます。
取得する情報や利用目的、第3者への提供などにつきましては、
以下の広告配信事業者様のプライバシーポリシーをご確認ください。
https://policies.google.com/technologies/ads?hl=ja
